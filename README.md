# Weather-App

A modern weather dashboard that provides real-time weather updates, hourly forecasts, and 3-day predictions with smart location search (city, village, or pincode). Built using HTML, Tailwind CSS, and JavaScript with WeatherAPI integration.

🌍 Smart location search (city, village, pincode)
🌡 Real-time weather data
⏱ Hourly forecast
📅 3-day forecast
📍 Current location detection
📱 Fully responsive design
